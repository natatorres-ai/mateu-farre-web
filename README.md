# Mateu Farre Web

Professional media kit / personal website for Mateu Farre.

## Project structure

- `index.html` contains the page structure.
- `styles.css` contains the custom visual design and animations.
- `script.js` contains language switching, dynamic cards and scroll interactions.
- `tailwind.config.js` contains the Tailwind CDN configuration.
- `assets/` contains the logo and images used by the site.

Open `index.html` directly in a browser. The site uses Tailwind via CDN and local images from `assets/`.
