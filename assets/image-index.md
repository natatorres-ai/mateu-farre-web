| Sección web | Imagen recomendada |

| Quién es Mateu | `Línea de equipo` |

| Galería | Mezcla de acción, equipo, celebración y cabina |

## Nota

Las imágenes de campeonato parecen muy potentes, pero antes de usarlas como “logro” conviene confirmar exactamente qué campeonato fue. Podemos usarlas visualmente sin afirmar datos no verificados.
